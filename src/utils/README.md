# 🛠️ Utilitários (`/utils`)

Funções auxiliares reutilizáveis, formatações genéricas e lógicas isoladas que não pertencem a um domínio específico de negócio.

## Responsabilidades

- **Formatadores**: Funções para formatar datas, moedas ou textos.
- **Criptografia / Hashes**: Lógica extra de hashing (ex: bcrypt helpers).
- **Geradores de Token**: Utilitários para criação rápida de tokens JWT.
- **Parsers**: Tratamento de objetos genéricos ou adaptações de payloads.
