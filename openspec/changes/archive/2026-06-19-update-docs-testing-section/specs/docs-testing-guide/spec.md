## ADDED Requirements

### Requirement: Documentação de Testes e Qualidade
O projeto MUST conter um documento dedicado em `docs/Testes_e_Qualidade.md` explicando todos os comandos de teste, ferramentas utilizadas e como executar e interpretar os resultados.

#### Scenario: Desenvolvedor quer rodar os testes
- **WHEN** um novo desenvolvedor acessa `docs/Testes_e_Qualidade.md`
- **THEN** ele encontra os comandos `npm test` e `npm run test:api` documentados com explicações claras, pré-requisitos (porta 3001) e como visualizar o relatório HTML.
